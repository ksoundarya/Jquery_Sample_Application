JQuery UI is one of the most successful UI libraries available today. JQuery UI is built with the JQuery Framework, by JQuery, and is without a doubt the most popular JavaScript library in use today. JQuery UI features multi-browser support which means peace of mind that someone using Internet Explorer, Firefox, Safari, or Chrome isn't going to leave your website with a sad face.
You are learning to use jQuery UI Widgets which is developed by "John Resig".
jQuery includes the following features!!!
  • DOM element selections using the multi-browser open source selector engine Sizzle, a spin-off of the jQuery project
  • Ajax
  • Events
  • JSON Parsing
  • Utilities, such as feature detection
  • Tooltip, Menus, Datepicker, Button and Other Theme Elements
  • its light weight when compared to other javascript frameworks.
  • it has a wide range of plugins available for various specific needs.
  • it is easier for a designer to learn jQuery as it uses familiar CSS syntax. jQuery is Javascript for Designers.
